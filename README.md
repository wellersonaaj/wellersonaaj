## Side projects

I build side projects to explore new product concepts, AI systems, marketplaces, integrations, and human-computer interfaces.

### Second Brain

A Telegram-first personal knowledge and task management system that captures everything I send and transforms fragmented information into useful context.

The system:

* Captures messages through Telegram
* Preserves every original message
* Uses LLMs to interpret and organize information
* Identifies connections between messages, people, projects, and topics
* Retrieves relevant memories and context when needed
* Converts information into tasks and follow-ups
* Connects with external tools through integrations

I designed it around a core principle: AI-generated interpretations should remain connected to their original sources, allowing the system to add intelligence and structure without losing the raw information.
